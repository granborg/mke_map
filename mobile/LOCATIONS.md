# Simulator location commands

Copy-paste `simctl` commands for testing the app in the iOS Simulator. `booted`
targets whichever simulator is running. Every coordinate below is a verified
interior point of its neighborhood — the app will resolve it to exactly the
name shown.

Generated by a script that computes an interior point per polygon in
[assets/data/milwaukee.json](assets/data/milwaukee.json); regenerate after
changing the GeoJSON or tier assignments.

```sh
# stop any route simulation / clear the fake location
xcrun simctl location booted clear
```

## Drive routes: good area → bad area

Simulated movement between waypoints. `--speed` is meters/second (30 ≈ 65 mph
driving, 2 ≈ walking); `--distance=25` issues a location update every 25 m,
which keeps the app's 50 m watcher well fed. The warning alert fires on entry
to any High/Highest neighborhood (once per neighborhood per 30 minutes).

### Downtown to the near north side

The classic demo. Starts lakefront-calm, ends deep red.

Historic Third Ward (tier 1) → Kilbourn Town (tier 3) → Triangle (tier 5) → Midtown (tier 5) → Walnut Hill (tier 5)

```sh
xcrun simctl location booted start --speed=30 --distance=25 43.0307,-87.9023 43.0423,-87.9183 43.0561,-87.9331 43.0525,-87.9465 43.0547,-87.9546
```

### East side to Harambee

Short hop across the river — Lowest straight into Highest, good for testing a single abrupt transition.

Upper East Side (tier 1) → Riverwest (tier 3) → Harambee (tier 5)

```sh
xcrun simctl location booted start --speed=30 --distance=25 43.0719,-87.8799 43.0718,-87.8983 43.0712,-87.9132
```

### Bay View to Clarke Square

South side run through a Moderate buffer into two warning-tier neighborhoods back-to-back.

Bay View (tier 1) → Lincoln Village (tier 3) → Historic Mitchell Street (tier 4) → Clarke Square (tier 4)

```sh
xcrun simctl location booted start --speed=30 --distance=25 42.9974,-87.8936 43.001,-87.9243 43.0117,-87.9248 43.0206,-87.9403
```

### Washington Heights to Metcalfe Park

West side, passing through Washington Park (Highest) on the way — expect two warnings.

Washington Heights (tier 1) → Washington Park (tier 5) → Metcalfe Park (tier 5)

```sh
xcrun simctl location booted start --speed=30 --distance=25 43.0547,-87.9797 43.0547,-87.9649 43.0642,-87.9542
```
